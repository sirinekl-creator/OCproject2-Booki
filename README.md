Projet 2 – Booki
Formation Développeur Web (Diplôme de niveau 5 (bac +2)) – OpenClassrooms

Contexte:
Dans le cadre de ma formation, je débute une alternance en tant que développeuse web au sein de la start-up Booki.
L’entreprise souhaite développer un site web permettant aux utilisateurs de trouver des hébergements et des activités dans la ville de leur choix.

Enjeux:
L’objectif de ce projet est d’intégrer une interface responsive à l’aide de HTML et CSS, en respectant une maquette fournie via Figma.

Compétences acquises à l'issu du projet:
- Implémenter une interface responsive avec HTML et CSS
- Intégrer du contenu conformément à une maquette avec HTML et CSS
- Versionner son projet avec Git et Github
- Installer un environnement de développement front-end
